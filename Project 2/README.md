# DecodeLabs Internship – Project 2: Exploratory Data Analysis (EDA)

## Project Overview
This project is part of the DecodeLabs Data Analytics Internship. The objective was to perform Exploratory Data Analysis (EDA) on a cleaned e-commerce dataset to uncover patterns, identify outliers, examine variable relationships, and generate business insights.

## Objectives
- Explore the structure of the dataset
- Generate descriptive statistics
- Analyze data distribution
- Detect outliers using the IQR
- Examine relationships between numerical variables using correlation analysis
- Draw business insights from the findings

## Tools Used
- Python
- Pandas
- Matplotlib
- Microsoft Excel
- GitHub

## Dataset
The analysis was performed using the cleaned dataset produced in Project 1 (Data Cleaning).

## Exploratory Analysis Performed
- Dataset overview
- Descriptive statistics
- Distribution analysis (Histogram)
- Boxplot visualization
- Five-number summary
- IQR outlier detection
- Correlation matrix

## Key Findings
- TotalPrice follows a right-skewed distribution.
- Eight high-value outliers were identified using the IQR method.
- UnitPrice showed the strongest positive correlation with TotalPrice.
- Quantity had a moderate positive relationship with TotalPrice.
- Most transactions were low to moderate in value, with only a few large purchases.

This project demonstrates how Exploratory Data Analysis transforms cleaned data into meaningful insights that support informed business decisions. The findings provide a foundation for further predictive analytics and data-driven decision-making.
